# BiteBack 🍽️  
### Save Food. Save Money. Save the Planet.

BiteBack is a Progressive Web App (PWA) created for **Technovation Girls 2026**.  
It helps families reduce food waste by tracking food expiry dates and showing which items are safe, expiring soon, or expired.

Food waste is a major cause of climate change because wasted food produces greenhouse gases when it decomposes in landfills. BiteBack encourages users to use food on time and reduce waste. The app also provides links to Irish organisations that support people facing food poverty.

## ⭐ Features

- Add food items with expiry dates
- Automatically shows days left until expiry
- Labels food as **Safe**, **Expiring Soon**, or **Expired**
- Delete food items from the list
- Impact Dashboard showing estimated CO₂ reduced and money saved
- Support & Donate section with Irish food poverty organisations
- Works on mobile and can be installed as a PWA app

## 📱 How to Install as a Mobile App (PWA)

### Android (Chrome)
1. Open the website link
2. Tap the 3 dots menu
3. Select **Add to Home Screen**
4. Install BiteBack

### iPhone (Safari)
1. Open the website link in Safari
2. Tap the Share button
3. Select **Add to Home Screen**
